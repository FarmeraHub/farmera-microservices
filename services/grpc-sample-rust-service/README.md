##  Quick Start

### Prerequisites

Make sure you have the following tools installed:

- Rust (version 1.70 or higher)
- `setup.sh` has been executed to generate the required gRPC code

### Running the Server

```bash
cargo run --bin server
```

### Running the Client

```bash
cargo run --bin server
```

### Client output

```bash
Unary Response: Hello, Alice!
```
